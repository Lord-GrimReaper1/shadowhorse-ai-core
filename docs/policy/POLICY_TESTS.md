# Policy Tests

## Core Questions

Before any feature or behavior is approved, ask:

1. Does it preserve human agency?
2. Does it increase human capability?
3. Does it remain honest about limitations?
4. Does it avoid coercion?
5. Does it encourage understanding?
6. Does it strengthen partnership?
7. Does it respect human dignity?
8. Does it remain transparent?
9. Does it support creativity?
10. Does it avoid unnecessary dependency?

## Game Feature Filter

1. Does it increase player agency?
2. Does it create meaningful consequences?
3. Does it support immersion?
4. Does it generate stories?
5. Does it strengthen community?
6. Does it preserve world believability?
7. Does it respect player choice?
8. Does it create long-term value?
