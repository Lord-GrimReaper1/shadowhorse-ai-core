# Pearl Media Workflows

Store approved ComfyUI workflow JSON files here.

Recommended starter layout:

- `image/concepts/` for concept art and mood board workflows.
- `image/assets/` for texture, material, prop, and reference workflows.
- `video/ltx/` for LTX-Video workflows.
- `render/` for deterministic render or turntable workflow notes.

Do not commit generated media, model checkpoints, or large downloaded assets here.
Generated outputs should go to the configured `COMFY_OUTPUTS_DIR`.
