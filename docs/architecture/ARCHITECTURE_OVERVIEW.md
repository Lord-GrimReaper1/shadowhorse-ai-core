# Architecture Overview

## System Shape

Shadowhorse AI Core is organized as a multi-agent framework with a human-led governance layer.

## Responsibilities

- Human layer: vision, values, creativity, direction, final approval.
- Orchestration layer: context management, routing, synthesis, memory continuity.
- Specialist layer: domain-focused reasoning and implementation.

## Specialist Roles

- Systems architect and continuity.
- Ethics and deep reasoning.
- Geography and multimedia synthesis.
- Exploration and discovery.
- Implementation and engineering.

## Principle

Specialization is preferred over generalization.
